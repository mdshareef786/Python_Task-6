# 🐍 Python OOP Practice Tasks

## 📌 Overview
This project contains Python OOP (Object-Oriented Programming) practice tasks that cover all core concepts with simple examples.

It is designed for beginners to intermediate learners to understand OOP step by step and apply it in real-world scenarios.

---

## 🧠 Concepts Covered

🔒 Encapsulation
- Private variables using (__)
- Getter and Setter methods
- Data hiding

🧬 Inheritance
- Single Inheritance (User → Student)
- Multilevel Inheritance (Faculty → TempFaculty)
- Code reusability

🔁 Method Overriding
- Same method name with different behavior
- Runtime polymorphism

🔗 Method Chaining
- Returning self from methods
- Calling multiple methods in one line

⚡ Polymorphism
- Same method call, different outputs

🧮 Class Variables
- Shared variable (users_count)
- Tracks number of objects created

---

## 📂 Tasks Included

✅ Task 1: Encapsulation
- Create User class
- Store username and password privately
- Implement register and login methods

✅ Task 2: Inheritance
- Parent class: User
- Child classes:
  - Student
  - Faculty
  - TempFaculty

✅ Task 3: Method Overriding
- Override greet() method in child classes

✅ Task 4: Method Chaining
Example:
user.login().greet().register()

✅ Task 5: Mini User System
- Combines all OOP concepts:
  - Encapsulation
  - Inheritance
  - Overriding
  - Method Chaining
  - Class Variable
